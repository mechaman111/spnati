Overview

Metadata defines characteristics about the character that will assist players in finding your character on the selection screen. This includes their name and selection screen image along with information for the filters, such as their size, gender, tags, and so forth.

Demographics

  • Label: This is the name other characters will use when referring to your character. This can be changed in the middle of the game.
  • First name: Character's first name during player selection. Not used anywhere else.
  • Last name: Character's last name during player selection. Not used anywhere else.
  • Gender: Character's gender. Characters react differently based on gender.
  • Chest/Crotch (depends on gender): For males, this is the size of their genitals. For females, this is the size of their chest. Characters react differently based on size.
  • Portrait: Which image to use on the player selection screen.
  • Height: How tall the character is. This data is currently useless.
  • Description: Character bio used in the character selection screen.
  • Origin: Source material (game, anime, movie, etc.)

Credits

  • Writer: Who wrote the dialogue for the character
  • Artist: Who made the art for the character

Credits

  • Intelligence: Determines how well the character plays. For player enjoyment's sake, you should try to balance AI and number of clothing layers. Bad AI with too few layers will lose too quickly too often, and good AI with too many layers will be frustrating to try to beat.
    You can specify different AI levels per character stage. A level will be used until the next listed stage is reached. For example, if you have two rows, 0-bad and 4-good, then your character will start with bad AI, but at stage 4, they will start using good AI.
    • throw: The character purposely tries to lose.
    • bad: The character randomly trades cards.
    • average: The character makes a decent effort to build a good hand. The majority of characters use this level.
    • good: The character goes for what's most probable to win a hand (without resorting to cheating).
    • best: Very similar to good, but slightly better.
  • Phases to finish: After a player has lost a hand while naked, they must masturbate. This value controls how many times the player has to click buttons before the character "finishes."
