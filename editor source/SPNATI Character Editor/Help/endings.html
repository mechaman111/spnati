Overview

Characters can optionally have one or more epilogues, which will be available to view the player wins a game against this character. Endings are like a non-interactive visual novel, consisting of a series of pictures with speech bubbles overlaid on top.
The Character Editor will not help in creating pictures for an epilogue. Instead, it helps with sequencing previously created pictures and positioning their speech bubbles. Common tools for creating the pictures are Kisekae with its built-in screenshot feature and/or Photoshop.

Creation

If your character has no endings yet, the Epilogue tab will be a black screen with everything disabled. Otherwise it will open the first ending. Use the dropdown in the upper left to switch between existing endings, or to create a new ending, click Add.
  • Available to gender: This controls which gender the player must be to be able to access the ending in game.
  • Title: This is the title that will display in the game when the player is given a choice of endings to view.

Screens

An ending is divided into multiple screen. Each screen consists of a picture and a series of speech bubbles that appear one after the other when the player clicks the Next button.

The middle of the Epilogue tab is the screen preview. This will give you a preview of the current screen as it would appear in game. You can resize the editor to see how your scene scales with various window sizes.

The bottom of the Epilogue tab is the screen editor, which consists of two parts.

  • Navigation Bar
    This lets you move between screens.
    1. Previous: Moves back to the previous screen.
    2. Image: Displays the file name for this screen's image.
    3. Set Image: Allows you to pick the file to use for this screen's image.
    4. Add (or Next): If you are on the last screen, this will add another screen. Otherwise it will advance to the next screen. When you create a new ending, the first screen is added automatically.
    5. Screen Number: This shows which screen you are editing currently, as well as how many screens there are total.
    6. Insert Screen: Inserts a screen at the current screen position. If you are at screen 2 of 3, for example, inserting will add a new screen 2, moving the old screen 2 to screen 3 (and screen 3 to 4).
    7. Remove Screen: Removes the current screen.
  • Speech Editor

    Each screen can have multiple speech bubbles, which appear in the ordered listed in this table. Every time the player clicks Next in game, the next bubble will appear. After the last bubble is displayed, Next will advance to the next screen.

    • X: Percentage from the left edge of the screen that the bubble starts (ex. 20% of the screen's width from the left).
    • Y: Percentage from the top edge of the screen that the bubble starts.
    • Width: Bubble's width in terms of screen width. The bubble's height is determined automatically by its text.
    • Arrow: Which side of the speech bubble that the bubble arrow should appear on.
    • Text: Text to put inside the bubble.
